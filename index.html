Project - Robot Car

This is likely my favourite project that I have made to date, created in 2019. All physical assembly and code were created entirely by myself. This project was powered by an “Arduino Due” microcontroller and the other main parts were 2 brushed motors, multiple optical light sensors and a servo motor. The first version of the car used light sensor readings to allow the robot car to follow a black line on the ground. Physical and code adjustments took quite some time, but I was able to get the car to a point where it would reliably follow the line every time. I added LEDs beside the optical light sensors which allowed for the car to operate as normal even in full darkness. The second version of the car involved upgrading to a sturdier body and the addition of a servo motor controlled robotic arm programmed for picking up and dropping off items. This version of the car went through tens of different arm designs until I was able to get it to reliably pick up and drop off a golf ball every time.

Project - LED Cube

This project was also entirely physical built and coded by me. This is a 4x3x3 LED Cube that I soldered together with individual LEDs and uses an “Arduino Due” microcontroller to display different patterns. Depending on which port on the Arduino was powered the LED Cube would display 1 of 4 different patterns. One being snake, scanning lines, diagonal scanning lines, or the letters P H D. It also contained a fifth option which would allow the user to adjust 3 potentiometers values to move 1 pixel’s X, Y, and Z values on a 3 dimensional plane.

Project - LCD Screen

This project again was entirely physical built and coded by myself. Using a triple digit 7 segment LCD screen I soldered every pin to a wire which would go through 2 different logic chips and then to an “Arduino Due” microcontroller. Once programmed, this allowed the Arduino to control every segment on the screen to display digits or numbers.
